# 一、实验目标
1. 了解Android应用中各种资源的概念与使用方法；
2. 掌握在Android应用中使用图片等资源的方法
# 二、实验内容
1. 在自选题目的项目中添加图片；
2. 引入对于图片并展示
# 三、实验步骤
1. 准备好一张图片，名为logo.jpg
2. 放入项目的res/drawable文件目录下；
3. 在activity_main.xml加入ImageView控件，在控件中引用该图片
```
android:src="@drawable/logo"
```
