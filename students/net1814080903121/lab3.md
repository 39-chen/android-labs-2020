# 一、实验目标 #

1. 了解Android应用中各种资源的概念与使用方法
2. 掌握在Android应用中使用图片等资源的方法。

# 二、实验内容 #

1. 在界面上显示至少一张图片（按照自己的题目添加）
2. 提交res/drawable及图片使用的代码
3. 将应用运行结果截图

# 三、实验步骤 #

1. 将要导入的图片放入res资源中的drawable中
2. 在layout对应的xml文件中使用ImageView对图片进行导入
